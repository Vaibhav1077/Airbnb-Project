# 🏡 Airbnb Clone (MERN Stack)

A full-stack **Airbnb Clone** web application developed using the **MERN Stack**.  
This project allows users to explore property listings, create accounts, and manage listings with image uploads and map integration.

---

## 🚀 Tech Stack

### Frontend
- React.js
- Vite
- HTML5, CSS3, JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB (Mongoose)

### Tools & Services
- Cloudinary (Image Uploads)
- Mapbox (Maps & Location)
- Passport.js (Authentication)
- Express Sessions
- EJS (Templating)

---

## ✨ Features

- User Authentication (Register / Login / Logout)
- Create, Edit, and Delete Listings
- Image Upload with Cloudinary
- Interactive Maps using Mapbox
- Secure Authentication & Authorization
- RESTful API Architecture
- Responsive UI
